# WSCNet_train

This project is an extension of Project [WSCNet2](https://github.com/Loyage/WSCNet2), as an example of training a droplet classification recognition network.

## Environment setup

1. Install Anaconda and create a python environment, then activate it.

2. ```
   conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia
   pip install opencv-python ipykernel tensorboard
   ```
